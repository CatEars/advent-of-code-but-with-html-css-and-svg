Advent of Code 2021 HTML + CSS Solution #2 - Part 1

Instructions

  1. Zoom out until the full width of the squares are shown
  2. Open Dev Tools / Inspector
  3. Calculate the volume of the layout of each square using the box model
    • Firefox: The box model is found by selecting the element and going to the sub-tab "Layout"
    • Chrome: Doesn't seem to show the correct size. Located under "Computed" tab in the inspector.
    • Edge: Same instructions as Chrome. But it shows the correct size. ¯\_(ツ)_/¯
  4. Subtract the volume of the green-line square (smaller at bottom) from the volume of the red-lined square (bigger at top)
